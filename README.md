- NPM or Yarn
- WhatsApp account
- 512MB+ RAM (recommended)
- Stable internet connection

## 🛠️ **Quick Setup**

### 1️⃣ **Fork & Clone**

<div align="center">
  <a href="https://github.com/RayBen445/CS-Assistant-/fork">
    <img src="https://img.shields.io/badge/Fork_Repository-black?style=for-the-badge&logo=github&logoColor=white" alt="Fork"/>
  </a>

```bash
git clone https://github.com/your-username/CS-Assistant-.git
cd CS-Assistant-
npm install
```

### 2️⃣ **Get Session ID**

<div align="center">
  <a href="https://cs-assistant-session.onrender.com/">
    <img src="https://img.shields.io/badge/Get_Session_ID-25D366?style=for-the-badge&logo=whatsapp&logoColor=white" alt="Get Session"/>

### 3️⃣ **Configure Environment**

```bash
cp .env.example .env
# Edit .env file with your SESSION_ID and other settings
```

### 4️⃣ **Run the Bot**

```bash
npm start
```

## 🔧 **Configuration**

### **Basic Settings**
```env
BOT_NAME=CS-Assistant
PREFIX=.
OWNER_NUMBER=your_number
TIME_ZONE=Africa/Johannesburg
```

### **Features**
```env
AUTO_READ_STATUS=false
WELCOME_MESSAGE=true
ANTI_LINK=false
ALWAYS_ONLINE=false
```

### **Security**
```env
ANTI_BAD_WORD=false
ONLY_GROUP=false
ONLY_ADMIN=false
```

## 🐛 **Troubleshooting**

### **Common Issues**

<details>
<summary><strong>Bot not responding to commands</strong></summary>

- Check if the prefix is correct (default: `.`)
- Ensure the bot has the necessary permissions
- Verify your SESSION_ID is valid
- Check the bot's connection status

</details>

<details>
<summary><strong>Session expired errors</strong></summary>

- Generate a new SESSION_ID
- Clear the session folder
- Restart the bot
- Re-scan the QR code

</details>

<details>
<summary><strong>Deployment fails</strong></summary>

- Check all environment variables are set
- Verify your GitHub repository is public
- Ensure you have the correct buildpacks
- Check the deployment logs for errors

</details>
